# Multi-Camera Human Detection with Telegram Alerts

This project monitors multiple cameras (USB / IP / RTSP) and detects humans using
YOLO (or HOG as fallback). When a person is detected, a photo is sent to Telegram
with basic event information.

## Features
- Monitor up to 70 cameras
- Human detection (YOLOv8, HOG fallback)
- Telegram photo alerts with cooldown
- Work-hours schedule (optional)
- Region-based detection (per camera)
- Automatic reconnect for unstable streams

## Tech Stack
Python, OpenCV, Ultralytics YOLO, Tkinter, Telegram Bot API
